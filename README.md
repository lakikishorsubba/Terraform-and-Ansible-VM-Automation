# Terraform + Ansible VM Automation

**Project Type:** DevOps Automation – Virtual Machine Provisioning and Configuration  

---

## Overview
**(Resource Automation & Infrastructure Management)** automates the **lifecycle of virtual machine provisioning and configuration** using **Terraform** and **Ansible**.  

## Objectives

1. **Infrastructure as Code (IaC):**  
   Learn how Terraform defines and manages infrastructure through declarative configuration and state management.

2. **Provisioning with Terraform:**  
   Automatically create **single or multiple Ubuntu VMs** in VirtualBox with custom CPU, memory, and network configurations.

3. **Terraform–Ansible Integration:**  
   Generate a **dynamic inventory file** from Terraform output for Ansible playbooks.

4. **Configuration Management with Ansible:**  
   Automate **Apache/Nginx installation**, service management, and deploy a **custom index.html** to verify success.

5. **Ansible Roles:**  
   Refactor playbooks into **modular roles** (tasks, handlers, templates, vars) for better structure and maintainability.

6. **Validation & Testing:**  
   Verify results using `terraform apply`, `ansible-playbook`, and `curl` to confirm end-to-end automation.
